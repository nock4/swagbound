Swagbound Atlas Browser

A searchable visual library for the tiles, characters, backgrounds, UI assets, town maps, motifs, buildings, named places, rooms, and interactables available to custom map authors.

Run pnpm dev, then open /atlas/. JSON and images load from this folder with relative URLs.

Overview

Headlines for the atlas vocabulary currently copied into apps/game/public/atlas.

Tiles are 32px arrangements; Characters are NPC/enemy sprite groups with Swagbound skin names when available; Motifs are flora and prop composites; Buildings, Places, Rooms, and Interactables are extracted world-authoring reference pieces.

Tiles

Per-tileset sheets with 32px cell inspection.

Hover or click a 32px cell to inspect arrangement, solidity, foreground, palette, and usage.

Characters

Sprite group portraits with skin status, role guesses, and NPC usage.

Backgrounds

Battle background images with enemy-group usage and override status.

UI

Window frames, fonts with glyph counts, and generated icon records.

Town Maps

Region map images with per-region town-map icon counts.

Motifs

Flora and prop composites with instance counts, size, and advisory category.

Buildings

Building composites with footprints, instance counts, doors, and sample areas.

Places

Named buildings with confidence, door/interior coordinates, and occupants.

Rooms

Interior room composites with dimensions, sector, area, and door count.

Interactables

Object, switch, and person anchors with text pointers, world pixels, and areas.
