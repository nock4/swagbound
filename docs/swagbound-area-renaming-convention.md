# Swagbound Area Renaming Convention

Status: adopted planning direction, 2026-07-03.

## Rule

Main towns should sound like simple EarthBound-style places first. Lore-heavy language belongs in buildings, dungeons, offices, statuses, items, and NPC systems.

Use:
- cute/plausible town names
- ordinary civic or resort language
- sinister institutional subareas
- doxx/leak language for the identity-pressure mechanic

Avoid as main town names:
- Signal Town
- Proofside
- Relay Yard
- Public Version Plaza
- VPL Village
- Foursigned City
- The Dry Strip

## Current world-name spine

- Morningside — Onett analogue / Bosch hometown
- Postwick — Twoson analogue / second town, posting/mail/rumor without sounding too online
- Bluebell Village — Happy Happy Village analogue / cute cult town
- Dead Letter — Threed analogue / haunted-doxxed town
- Little Swag World — Saturn Valley analogue / LSW + Drifella sprites replace Mr. Saturn-style residents
- The Galleria — Fourside analogue / mall, market, fashion, city, institution
- Solana Beach — Summers analogue / resort vice zone
- Vacancy Flats — Dusty Dunes analogue / empty lots, heat, model homes, dead development
- The Unlisted Room — Magicant analogue / Bosch private inner space

## Concept rename

Retire `Public Version` as player-facing language. The clearer spine is doxxing/leaks:

- leaked version
- leaked Bosch
- Dox Sheet
- Doxx Clique
- doxxed
- leak
- listing
- Opt-Out Desk
- Suppression Center
- Unlisted

Runtime IDs/file paths may keep older stable handles such as `public-version-card` until a scoped ID migration is approved.
